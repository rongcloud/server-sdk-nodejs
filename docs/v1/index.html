Server SDK

各平台 SDK 源码、示例 请移步: Java、 Node

SDK 使用指南

图解

蓝色框表示应用服务器 (App Server)，用来处理业务数据，比如用户信息、群组关系等

App Server 会使用到融云 Server SDK： Java、 Node

蓝色为应用（App），客户端发送消息使用，会用到各端 SDK: iOS、Android、Web、Desktop

绿色框表示融云服务器 (RongCloud Server) ，是消息通道，调用各端 SDK 发送消息即可
